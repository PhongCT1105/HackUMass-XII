**Start backend server:**

`python3 -m venv venv`

`source venv/bin/activate`

`pip install -r requirement.txt` (ios => make sure there's no pywin32 in requirement.txt)

`uvicorn main:app --reload`

**Start frontend:**

`npm i`

`npm run dev`
